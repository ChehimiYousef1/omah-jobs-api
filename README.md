# 🚀 OMAH Jobs API

A production-ready RESTful API for managing job listings, authentication, and recruitment workflows — built with Node.js, Express, and SQL Server.

---

## 📌 Features

- 🔐 Authentication (JWT + OAuth: Microsoft / GitHub)
- 👤 User Management
- 💼 Job Management (CRUD)
- 📊 Secure Cookie-based Auth
- 📄 Swagger API Documentation
- ☁️ Deployment Ready (Render)

---

## 🛠️ Tech Stack

- Node.js / Express.js
- SQL Server (mssql)
- JWT Authentication
- Swagger (API Docs)
- Nodemailer (Email Service)
- OAuth (Microsoft / GitHub)
